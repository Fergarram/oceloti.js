How was this built?

This tool was made using Oceloti.js, a webroom framework that I created after prototyping a lot of infinite canvas tools.

Hey there,

Try to drag cards around and use the mouse wheel to pan through the room.

You can play around and download your room html file by hitting CTRL/CMD+S.

Have fun!

NOTE: You can upload your html file to are.na and host it that way :)
